# Shyama Public School Website

Final static website for Shyama Public School, Gyanpur, Bhadohi.

## Live Site

https://shyama-public-school-gyanpur.netlify.app

## Pages

- Home: `index.html`
- About Us: `about.html`
- Admission: `admission.html`
- Events: `events.html`
- Gallery: `gallery.html`
- Contact: `contact.html`
- Privacy Policy: `privacy.html`
- Terms: `terms.html`
- CBSE Mandated Disclosure: `disclosure.html`
- Error Page: `404.html`

## Features

- Modern responsive school website UI
- Mobile-friendly navigation
- Google Maps location on home and contact pages
- Gallery with filter and lightbox
- Admission and contact inquiry forms
- Call and WhatsApp quick-action buttons
- Sitemap, robots file, favicon, and Netlify configuration

## Deployment

This is a static frontend website. Deploy the repository root to any static host.

For Netlify, the included `netlify.toml` publishes the current folder.
