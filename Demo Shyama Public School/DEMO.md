# Shyama Public School Website Demo

This demo build contains the redesigned frontend for Shyama Public School, Gyanpur.

## Demo Pages

- Home: `index.html`
- About Us: `about.html`
- Admission: `admission.html`
- Events: `events.html`
- Gallery: `gallery.html`
- Contact: `contact.html`
- Privacy Policy: `privacy.html`
- Terms & Conditions: `terms.html`
- Mandatory Disclosure: `disclosure.html`

## Current Status

- Modern premium school UI applied.
- Responsive navbar, hero, cards, gallery, footer and buttons updated.
- Gallery duplicate/cache images removed from the page.
- Computer Lab, Transport and Sports Ground visuals added.
- Forms, gallery lightbox, carousel and card interactions are wired through `main.js`.
- Local preview was verified at `http://127.0.0.1:8001/index.html`.

## Notes

The site is a static frontend project. Open `index.html` directly or serve the folder with any static web server.
